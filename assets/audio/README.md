# CrystalPopperz — Audio Files

Place the following audio files in this `assets/audio/` folder:

| File           | Description                          | Format | Notes                         |
|----------------|--------------------------------------|--------|-------------------------------|
| bg_music.mp3   | Background music (loops)             | MP3    | Upbeat, casual puzzle style   |
| pop.mp3        | Crystal pop / match sound            | MP3    | Short (< 0.5s)                |
| swap.mp3       | Crystal swap sound                   | MP3    | Short (< 0.3s)                |
| win.mp3        | Level complete fanfare               | MP3    | 1–2 seconds                   |
| lose.mp3       | Game over sound                      | MP3    | Short (< 1s)                  |
| power.mp3      | Power crystal activation             | MP3    | Short burst (< 0.5s)          |

## Free Music Sources
- https://opengameart.org  (search: puzzle music, match3)
- https://freemusicarchive.org
- https://pixabay.com/music (filter: game / fun)

## Important
The app is built to gracefully handle missing audio files —
it will run silently if the MP3 files are not present.
No crash will occur. Add files when ready.
